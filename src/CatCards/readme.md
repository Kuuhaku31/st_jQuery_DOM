# 随机猫咪卡片大战

使用 JavaScript 和 JQuery 制作的网页小游戏

## 整体玩法逻辑

1. 开始
2. 随机抽卡诺干次获得初始卡牌
3. 检查场上是否存在敌方卡牌

  如果没有，电脑进行抽卡

4. 玩家决定一张出战卡牌
5. 开始对局
6. 双方卡牌进行对战

  对战逻辑：根据自身攻击力和对方防御力计算伤害，更新双方卡牌的生命值

  如果玩家卡牌生命值降为 0 或以下，玩家失败，失去出战卡牌

  如果电脑卡牌生命值降为 0 或以下，玩家获胜，获得回复生命值的电脑的卡牌

7. 如果玩家还有卡牌，返回步骤 3；否则游戏结束

## 技术细节

### 抽卡

使用 API:

- 随机生成猫咪图片: `https://api.thecatapi.com/v1/images/search`
- 随机生成用户信息: `https://randomuser.me/api/?results=3&nat=jp`

每次抽卡时，调用上述 API 获取随机猫咪图片和用户信息，并将它们组合成一张卡牌

每张卡牌包含以下属性：

- 图片 URL
- 猫咪名字（从用户信息中提取）
- 攻击力（从用户信息中的体重信息提取）

### 对战逻辑

玩家和电脑各自选择一张卡牌进行对战，比较攻击力：

- 如果玩家的攻击力高于电脑，玩家获胜，获得电脑的卡牌
- 如果玩家的攻击力低于电脑，玩家失败，失去出战卡牌

### 玩家对拥有的卡牌进行管理

所有卡牌同时显示在页面上，每张卡牌都有`出战`和`丢弃`按钮：

- `出战`按钮：将该卡牌设置为当前出战卡牌，进入对战阶段
- `丢弃`按钮：将该卡牌从玩家的卡牌列表中移除
